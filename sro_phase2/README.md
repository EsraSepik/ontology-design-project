# Smart Recipe Ontology (SRO) - Phase 2 Deliverables

Ontology version: 2.0  
Specification version: 2.0  
Prepared date: 2026-05-08

## Contents

- `ontology/smart-recipe-ontology-v2.ttl`: main Turtle ontology file.
- `ontology/smart-recipe-ontology-v2.owl`: RDF/XML serialization for Protege.
- `data/sample_recipe_data_phase2.csv`: small validation dataset for data acquisition and mapping.
- `queries/competency_queries.rq`: SPARQL queries mapped to SRO competency questions.
- `queries/query_results.txt`: query results executed over the Phase 2 ontology.
- `reports/SRO_Phase2_Report.docx` and `.pdf`: submission report.
- `reports/SRO_Specification_Document_v2.docx` and `.pdf`: updated ORSD/specification document v2.
- `docs/index.html`: documentation preview; replace with official Widoco output when Widoco is run.
- `README_GitHub_Widoco.md`: GitHub versioning and Widoco commands.

## Phase 2 summary

This version extends SRO from a basic recipe model into a richer recipe knowledge graph model. It adds n-ary ingredient usage, ordered preparation steps, nutrition profiles, diet tags, cuisine and course categories, kitchen utilities, data source/provenance, extraction confidence, and competency queries.
